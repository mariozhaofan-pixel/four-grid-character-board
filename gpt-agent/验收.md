# 画面验收参考

生成结果的视觉检查依据：

- 首要检查：结果必须是一张独立单图，不得包含拼图、四宫格、网格、分屏、多面板或多个视角。
- 输出是新生成的可见图片，而不是用户参考图的原样返回或轻度修改。
- 只有一位人物、一个视角和一个连续画面，符合用户要求的画面类型和构图。
- 人物身份、脸型、五官、肤色、发型、服装、鞋袜、配饰、道具与当前正面全身保持一致。
- 正面全身和背面全身为竖屏9:16；正脸特写和侧脸特写为1:1。
- 默认灰底；主体含明显灰色系时使用白底。
- 在不裁掉必要部位的前提下尽量减少留白。
- 无文字、水印、logo、标签、编号、边框、拼图、分屏、四宫格、角色设定页或多视角排版。
