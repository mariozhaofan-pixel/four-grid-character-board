# 验收、失败恢复与完成输出

本文件不参与正常生图提示词构造。只在当前图片返回、图像工具明确失败、用户要求验收或四张全部完成时读取。

## 完成状态判定

只有对话中真实返回的生成图片才算完成。文件名、路径、Markdown占位、空白附件、计划说明或“已生成”文字都不算完成。

## 单图验收

- 新生成图片，不是用户参考图的原样返回、改名、裁切、放大或轻度修饰。
- 只有一位人物，符合当前步骤的视角与构图。
- 身份、年龄、脸型、五官、肤色、发型、服装、鞋袜、配饰、道具与当前母图一致。
- 母图和背面全身图为竖屏9:16；两张头像为1:1。
- 默认灰底；主体含明显灰色系时为白底。
- 在不裁掉必要部位的前提下尽量减少留白。
- 两张头像保留毛孔、细纹、红润感、明暗变化、光影颗粒和灰尘噪点，除非用户明确要求相反效果。
- 无文字、水印、logo、标签、编号、边框、四宫格、角色设定页或多视角排版。

图片明显错误时，指出当前步骤的具体错误，停在该步骤并等待用户决定是否重做；同一回复不自动再次调用。

## 工具失败恢复

图像工具明确返回失败、不可用、报错或限流时：

1. 停在当前步骤，不改变完成状态。
2. 不自动重试，不声称图片已生成。
3. 不输出虚假路径或占位附件。
4. 输出当前步骤短恢复提示词。

```text
图像生成工具未返回图片。当前步骤：【步骤文件名 / 输出文件名】

恢复提示词：
【当前步骤文件中的参考图分工、构图、画幅、背景、质感和限制项组成的短提示词】
```

## 四张完成

固定四张为：

1. `master_front_full_body.png`
2. `image_01_front_portrait.png`
3. `image_02_three_quarter_portrait.png`
4. `image_03_back_full_outfit.png`

建议用户自行拼接：

- 左上：正面全身母图
- 右上：正面头像
- 左下：三分之二侧脸头像
- 右下：背面全身服装图

完成后列出真实已生成图片、建议拼接顺序和人物简单描述提示词；未实际生成的图片明确标记为未生成。

```text
人物简单描述提示词：
成年【人种/性别】，五官【简洁描述】，皮肤【简洁描述】，表情【简洁描述】。发型为【发色、长度、分缝、刘海、发丝状态、发饰】。身穿【服装类型、颜色、材质、关键结构、装饰细节、鞋袜/手套/道具】。
```
